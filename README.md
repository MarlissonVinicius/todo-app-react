## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com foco em aprendizado prático de:

- React Hooks (`useState`, `useNavigate`, `useSearchParams`)
- Manipulação de listas (`map`, `filter`)
- Componentização
- Tipagem com TypeScript
- Roteamento com React Router
- Estilização com Tailwind CSS

---

## 🚀 Funcionalidades

- Lista de tarefas dinâmicas
- Adição de novas tarefas
- Remoção de tarefas
- Marcar tarefas como concluídas
- Página de detalhes da tarefa
- Navegação entre páginas

---

## 🛠️ Tecnologias utilizadas

- ⚛️ React
- ⌨️ TypeScript
- ⚡ Vite
- 🎨 Tailwind CSS
- 🌐 React Router DOM
- 🎯 Lucide React (ícones)
